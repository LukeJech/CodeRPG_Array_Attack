Understanding Arrays/Lists

Skip the reading and head straight to the video lesson!

In programming, we can set variables equal to a number, string of words, and more.

However, we can also set a variable to be equal to a group of things. This is where Arrays, called Lists in Python, are very useful!

Video Game Inventory Example

Imagine creating an inventory for your player and having to do it like this...

inventory_item1 = 'sword'

inventory_item2 = 'potion'

inventory_item3 = 'ax of jave'

This of course is very tedious and instead we should use one variable as an array for the whole inventory...

inventory = ['sword', 'potion', 'ax of jave']

Now we can access any item in our inventory using what are called indexes.

An index is simply a number representing the position of our array. Only instead of our array starting with an index of 1, it starts with an index of 0. Let's say we want to access our sword...

Indexes = 0 1 2

inventory = ['sword', 'potion', 'ax of jave']

We can access the sword by using array syntax which uses square brackets [] with the index number inside.

inventory[0] would equal our string of 'sword'.

inventory[1] would equal our string of 'potion'.

Hopefully you can guess what inventory[2] would equal...
